Interveiw Qustions

Q1. What does HTML stand for?

Answer: HTML stands for HyperText Markup Language.

Q2. What is the purpose of the <head> tag in HTML?

Answer: The <head> tag contains meta-information about the HTML document, such as the title, character set, linked CSS files, and scripts.

Q3. What is the difference between a block-level element and an inline element?

Answer: Block-level elements occupy the full width available and start on a new line (e.g., <div>, <p>), while inline elements only take up as much width as necessary and do not start on a new line (e.g., <span>, <a>).

Q4. How can you include comments in HTML?

Answer: Comments in HTML are included using <!-- This is a comment -->.

Q5. What is the purpose of the <alt> attribute in the <img> tag?

Answer: The <alt> attribute provides alternative text for an image if it cannot be displayed. It improves accessibility for screen readers and provides context for images.

Q6. What are semantic HTML elements?

Answer: Semantic HTML elements clearly describe their meaning in a human- and machine-readable way. Examples include <article>, <section>, <header>, <footer>, and <aside>.

Q7. How do you create a hyperlink in HTML?

Answer: A hyperlink is created using the <a> tag with the href attribute, like so: <a href="https://www.example.com">Click here</a>.

Q8. What is the difference between the <div> and <span> tags?

Answer: <div> is a block-level element used for grouping content and layout purposes, while <span> is an inline element used for styling small chunks of content within a block.

Q9. What is the use of the <form> element?

Answer: The <form> element is used to create an HTML form for user input. It can contain various input elements like text fields, checkboxes, radio buttons, and submit buttons.

Q10. How can you embed a video in an HTML page?

Answer: You can embed a video using the <video> tag, like so: <video controls src="video.mp4"></video>.

Q11. What is the purpose of the <meta> tag in HTML?

Answer: The <meta> tag provides metadata about the HTML document, such as character encoding, viewport settings, and author information.

Q12. How do you create a table in HTML?

Answer: A table is created using the <table> tag, with rows defined by <tr>, headers by <th>, and data cells by <td>. Example: 
<table>
              <tr>
                <th>Header 1</th>
                <th>Header 2</th>
              </tr>
              <tr>
                <td>Data 1</td>
                <td>Data 2</td>
              </tr>
            </table>

Q13. What is the difference between <strong> and <b> tags?

Answer: The <strong> tag indicates that its contents have strong importance and is often displayed as bold, while <b> simply applies bold styling without implying any importance.

Q14. How do you specify the language of an HTML document?

Answer: The language of an HTML document is specified using the lang attribute in the <html> tag, like so: <html lang="en">.

Q15. What is the difference between the <input> types "text" and "password"?

Answer: The "text" type creates a single-line text input field, while the "password" type hides the text entered by the user for privacy.

Q16. How do you create a list in HTML?

Answer: HTML supports ordered lists (<ol>) and unordered lists (<ul>), with list items defined by <li>. Example: 
<ul>
              <li>Item 1</li>
              <li>Item 2</li>
            </ul>

Q17. What does the target="_blank" attribute do in a link?

Answer: The target="_blank" attribute makes the link open in a new browser tab or window.

Q18. How can you include external CSS in an HTML document?

Answer: External CSS is included using the <link> tag in the <head>, like so: <link rel="stylesheet" href="styles.css">.

Q19. What is the purpose of the id attribute in HTML?

Answer: The id attribute provides a unique identifier for an HTML element, which can be used for styling with CSS or manipulating with JavaScript.

Q20. What are data attributes in HTML?

Answer: Data attributes are custom attributes that store extra information about an element, prefixed with data-. For example: <div data-info="value"></div>.

Q21. What does CSS stand for?

Answer: CSS stands for Cascading Style Sheets.

Q22. What is the difference between class selectors and ID selectors in CSS?

Answer: Class selectors are denoted by a dot (.) and can be used multiple times in a document, while ID selectors are denoted by a hash (#) and should be unique within a document.

Q23. How do you center a block-level element horizontally in CSS?

Answer: You can center a block-level element horizontally using margin: 0 auto; and specifying a width.

Q24. What is the CSS box model?

Answer: The CSS box model describes the layout of elements as a box with margins, borders, padding, and content areas.

Q25. How do you apply styles to all elements of a certain type?

Answer: Styles can be applied to all elements of a certain type using element selectors. For example: p { color: red; } styles all <p> elements.

Q26. What are pseudo-classes in CSS?

Answer: Pseudo-classes are used to define the special states of an element. Examples include :hover, :focus, and :nth-child().

Q27. How do you include custom fonts in a web page?

Answer: Custom fonts are included using the @font-face rule in CSS. Example: 
@font-face {
              font-family: 'CustomFont';
              src: url('customfont.woff2') format('woff2');
            }
            body {
              font-family: 'CustomFont', sans-serif;
            }

Q28. What is the z-index property used for in CSS?

Answer: The z-index property controls the stacking order of positioned elements. Higher values are displayed in front of lower values.

Q29. How do you create a responsive design in CSS?

Answer: Responsive design is achieved using media queries to apply different styles based on the viewport size. Example: 
@media (max-width: 600px) {
              body {
                font-size: 14px;
              }
            }

Q30. What is the flexbox layout model?

Answer: Flexbox is a CSS layout model that allows for flexible and efficient layout designs. It helps in aligning items within a container and distributing space.

Q31. How do you use CSS Grid Layout?

Answer: CSS Grid Layout allows for a two-dimensional grid-based layout. It is defined with display: grid; and utilizes grid lines, rows, and columns to position elements.

Q32. What is the difference between padding and margin in CSS?

Answer: Padding is the space between an element's content and its border, while margin is the space outside the border of the element.

Q33. How do you apply styles only when a user hovers over an element?

Answer: You use the :hover pseudo-class to apply styles when an element is hovered over. Example: 
a:hover {
              color: red;
            }

Q34. What is the purpose of the position property in CSS?

Answer: The position property determines how an element is positioned in the document. Values include static, relative, absolute, fixed, and sticky.

Q35. How do you make an element fixed to the viewport?

Answer: Use position: fixed; to make an element fixed relative to the viewport. Example: 
.fixed {
              position: fixed;
              top: 0;
              left: 0;
            }

Q36. What are CSS transitions?

Answer: CSS transitions allow for smooth changes in property values over time, defined using the transition property. Example: 
.box {
              transition: background-color 0.5s;
            }
            .box:hover {
              background-color: blue;
            }

Q37. What are CSS animations?

Answer: CSS animations enable the creation of complex animations using the @keyframes rule and the animation property to define the behavior over time.

Q38. How can you ensure cross-browser compatibility in CSS?

Answer: Use vendor prefixes for experimental features, validate your CSS, and test across different browsers to ensure compatibility.

Q39. What is the box-sizing property used for in CSS?

Answer: The box-sizing property defines how the width and height of an element are calculated, including or excluding padding and border. Values include content-box and border-box.

Q40. How do you use CSS variables (custom properties)?

Answer: CSS variables are defined using --variable-name syntax and can be used with the var() function. Example: 
:root {
              --main-color: blue;
            }
            .element {
              color: var(--main-color);
            }

Q41. What is a CSS preprocessor, and name one example?

Answer: A CSS preprocessor extends CSS with variables, functions, and other features. Examples include SASS, LESS, and Stylus.

Q42. How do you clear floats in CSS?

Answer: Clearing floats can be done using the clear property, like so: clear: both;, or using a clearfix technique.

Q43. What is the clip-path property in CSS?

Answer: The clip-path property is used to create complex shapes and masks for elements, defining which parts of the element are visible.

Q44. How do you style form elements in CSS?

Answer: Form elements like <input>, <select>, and <textarea> can be styled using standard CSS properties, but they may require additional styling to ensure consistency across browsers.

Q45. What are media queries in CSS?

Answer: Media queries are used to apply different styles based on the viewport size, device orientation, or other conditions.

Q46. How do you create a responsive grid layout in CSS?

Answer: You can create a responsive grid layout using CSS Grid or Flexbox, with media queries to adjust the layout based on screen size.

Q47. What is the display property in CSS?

Answer: The display property defines how an element is displayed on the page, with values like block, inline, inline-block, none, and flex.

Q48. How do you add a shadow to text in CSS?

Answer: Use the text-shadow property to add a shadow effect to text. Example: 
.text {
              text-shadow: 2px 2px 4px rgba(0,0,0,0.5);
            }

Q49. What is the opacity property used for in CSS?

Answer: The opacity property controls the transparency of an element, with values ranging from 0 (completely transparent) to 1 (completely opaque).

Q50. How do you use the :nth-child() pseudo-class in CSS?

Answer: The :nth-child() pseudo-class targets elements based on their position in a parent element. Example: 
li:nth-child(odd) {
              background-color: #f0f0f0;
            }
